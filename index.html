Reise bearbeiten

☁️ Cloud-Sync einrichten

Einmalig einrichten — dann seht ihr automatisch die Einträge des anderen, ohne Link zu schicken. Kostenlos via Supabase, keine Kreditkarte.

Anleitung anzeigen (5 Min, kostenlos)
  1. Öffne supabase.com → oben rechts "Start your project".
  2. Mit GitHub oder E-Mail einloggen (kostenlos, keine Karte).
  3. "New Project" → Name: tripmate → DB-Passwort setzen → Region Europa wählen → "Create".
  4. ~1 Minute warten, bis das Projekt bereit ist.
  5. Links im Menü: "SQL Editor" → "+ New query" → folgenden Code einfügen, dann "Run": 
    create table trips (
  room_code text primary key,
  state jsonb,
  updated_at timestamptz default now()
);
alter table trips enable row level security;
create policy "r" on trips for select using (true);
create policy "i" on trips for insert with check (true);
create policy "u" on trips for update using (true);
  6. Links "Project Settings" (⚙) → "API". Dort kopieren:
    • Project URL (z.B. https://abcd.supabase.co)
    • anon public Key
  7. Felder unten ausfüllen + "Aktivieren" tippen.
  8. Deiner Frau gibst du: dieselbe URL, denselben Key und denselben Room-Code. Sie füllt sie hier ein — fertig, ab dann automatisch Sync.

Oder manuell einrichten
📖 Anleitung (5 Min, kostenlos)
  1. Öffne supabase.com → "Start your project".
  2. Mit GitHub oder E-Mail einloggen (kostenlos, keine Karte).
  3. "New Project" → Name: tripmate → DB-Passwort setzen → Region Europa → "Create".
  4. ~1 Min warten bis bereit.
  5. "SQL Editor" → "+ New query" → folgendes einfügen, dann "Run": 
    create table trips (
  room_code text primary key,
  state jsonb,
  updated_at timestamptz default now()
);
alter table trips enable row level security;
create policy "r" on trips for select using (true);
create policy "i" on trips for insert with check (true);
create policy "u" on trips for update using (true);
  6. "Project Settings" (⚙) → "API" → Project URL und anon public Key kopieren.
  7. Unten ausfüllen, "Aktivieren". Fertig – schick Jane den Einladungs-Link!
